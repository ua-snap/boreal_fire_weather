# boreal_fire_weather
Produce CFFDRS fire weather indices from bias-corrected CMIP6 and analyze wildfire in boreal ecoregions. 

This repo revises and reorganizes Adam Young's code (https://zenodo.org/records/7783759) that was used to produce this data release: https://www.sciencebase.gov/catalog/item/67ead89cd34ed02007f8357f.
